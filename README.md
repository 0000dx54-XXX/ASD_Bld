# 🎮 ASD_Bld: CS3 Android Installer

Утилита для автоматизированной установки **Counter-Strike 3 (Alpha)** на устройства Android. 

> [!CAUTION]
> **Статус установщика:** Beta.  
> **Версия игры:** Alpha.  
> **Важно:** Для входа в игру потребуется **Google Account** для авторизации.

## 🚀 Особенности
*   **Проект:** ASD_Bld
*   **Движок установщика:** DevelNext (PHP)
*   **Метод работы:** Установка через **Отладку по Wi-Fi (Wireless Debugging)**.

## 📱 Системные требования
*   **ОС:** Android 11, 12, 13, 14 или **самая последняя версия**.
*   **Процессор:** Рекомендуется **самый мощный Snapdragon** (серии 8 Gen 1 / Gen 2 / Gen 3 и выше) для стабильного FPS.
*   **Аккаунт:** Google Account для входа в игру.
*   **Сеть:** Стабильное Wi-Fi соединение (ПК и телефон в одной сети).

## 🛠 Инструкция по подготовке

### 1. Как включить "Режим разработчика"
1. Зайдите в **Настройки** -> **О телефоне**.
2. Нажмите на **"Номер сборки"** (Build Number) **7 раз** подряд.
3. Введите пароль устройства, если потребуется.

### 2. Включение отладки по Wi-Fi (Android 11+)
1. Перейдите в **Настройки** -> **Система** -> **Для разработчиков**.
2. Включите **Отладку по USB**.
3. Включите **Отладку по Wi-Fi** (Wireless Debugging).
4. Переключите тумблер **"Wifi Debug"** в программе `ASD_Bld` и нажмите **"Старт"**.
