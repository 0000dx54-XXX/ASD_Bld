# 🎮 CS3 Android Installer (Beta)

Утилита для автоматизированной установки Counter-Strike 3 на устройства Android. 

> [!WARNING]  
> Проект находится на стадии **Beta-тестирования**. Возможны ошибки и нестабильная работа.

## 🚀 Особенности
*   Установка CS3 в один клик.
*   Поддержка актуальных мобильных билдов.
*   **Важно:** На текущем этапе установка и настройка производятся строго через **отладку по Wi-Fi (Wireless Debugging)**.

## 🛠 Инструкция по установке

Для работы программы необходимо активировать режим разработчика на вашем смартфоне:

1.  Зайдите в **Настройки** -> **Для разработчиков**.
2.  Включите **Отладку по USB** и **Отладку по Wi-Fi**.
3.  Подключитесь к той же сети Wi-Fi, что и ваш компьютер.
4.  Запустите установщик и следуйте инструкциям в консоли.

## 📋 Системные требования
*   Android 10.0 или выше.
*   Стабильное Wi-Fi соединение.
*   Свободное место: ~5 ГБ.
