# 🎮 ASD_Bld: CS3 Android Installer

Утилита для автоматизированной установки **Counter-Strike 3 (Alpha)** на устройства Android. 

> [!CAUTION]
> **Статус установщика:** Beta.
> **Версия игры:** Alpha (возможны баги и вылеты).

## 🚀 Особенности
*   **Проект:** ASD_Bld
*   **Цель:** Быстрая установка раннего билда CS3.
*   **Метод работы:** Установка через **Отладку по Wi-Fi (Wireless Debugging)**.

## 🛠 Инструкция (Отладка по Wi-Fi)

Для успешной работы `ASD_Bld` выполните следующие шаги:

1.  **На смартфоне:** зайдите в *Настройки* -> *Для разработчиков*.
2.  Включите **Отладку по USB** и **Отладку по Wi-Fi**.
3.  Убедитесь, что телефон и ПК в одной сети Wi-Fi.
4.  Запустите `ASD_Bld` и следуйте инструкциям на экране для сопряжения устройств.

## 📋 Техническая информация
*   **Версия установщика:** 0.1-beta
*   **Версия контры:** Alpha Build
*   **Требования:** Android 10+, стабильный Wi-Fi.
