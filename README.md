# 🎮 ASD_Bld: CS3 Android Installer

Утилита для автоматизированной установки **Counter-Strike 3 (Alpha)** на устройства Android. 

> [!CAUTION]
> **Статус установщика:** Beta.
> **Версия игры:** Alpha.
> **Важно:** Для входа в игру потребуется **Google Account** для авторизации.

## 🚀 Особенности
*   **Проект:** ASD_Bld
*   **Метод работы:** Установка через **Отладку по Wi-Fi (Wireless Debugging)**.

## 🛠 Инструкция по подготовке

### 1. Как включить "Режим разработчика"
Если это меню у вас еще не открыто:
1. Зайдите в **Настройки** -> **О телефоне**.
2. Найдите пункт **"Номер сборки"** (Build Number).
3. Нажмите на него быстро **7 раз** подряд.
4. Вы увидите сообщение: *"Вы стали разработчиком!"*.

### 2. Включение отладки по Wi-Fi
1. Перейдите в **Настройки** -> **Система** (или Дополнительно) -> **Для разработчиков**.
2. Включите переключатель **Отладка по USB**.
3. Включите переключатель **Отладка по Wi-Fi** (на некоторых устройствах нужно подтвердить подключение в той же сети).

### 3. Авторизация
После установки при первом запуске игры вам необходимо будет войти в свой **Google Account**, чтобы пройти проверку лицензии Alpha-теста.
